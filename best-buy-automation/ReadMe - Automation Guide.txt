Automating Basic Flows for Best Buy Site: A Project Overview
	This project focuses on the automation of essential flows for the Best Buy site. The automation of these flows is intended to streamline and expedite user interaction with the site, ultimately improving user experience.

There 2 packages:
1. com.best.buy.pre.set.flows   2. com.best.buy.user.based.flows

	com.best.buy.pre.set.flows:
		This package includes pre-set flows, with predefined values, that will execute automatically.
	
	com.best.buy.user.based.flows
		The second package involves flows that require user interaction. To proceed with these flows, users will be prompted to provide values via the console.
